## ATM Withdrawal

## Pré-requisitos

[Node.js](https://nodejs.org/en/download)
&nbsp;

[Docker](https://www.docker.com/get-started/)

## Como executar o projeto

### 1. Clonando o projeto e instalando dependências:

```
git clone https://github.com/darkclyan/atm.git
```

```
cd atm
```

```
npm install
```

### 2. Iniciando o projeto

#### Opção A: Usando Docker

```
cd atm
```

```
npm run dev:install
```

```
npm run dev
```

#### Opção B: Usando Node

##### Back-end

```
cd ./atm/atm-api
```

```
npm install
```

```
npm run start
```

##### Front-end

```
cd ./atm/atm-app
```

```
npm install
```

```
npm run start
```

## Contribuindo para Nome do Projeto

### Faça um fork deste repositório.

Crie uma branch:

```
git checkout -b nome-da-branch
```

Faça suas alterações e confirme-as:

```
git commit -m 'mensagem-do-commit'
```

Envie para a branch original:

```
git push origin atm/nome-da-branch
```
